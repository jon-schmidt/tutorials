# Tutotials - NGINX

## Controlling nginx
After starting send a signal to NGINX with:[0]

> nginx -s *signal*

*signal* options:
- `stop` — fast shutdown
- `quit` — graceful shutdown
- `reload` — reloading the configuration file
- `reopen` — reopening the log files
